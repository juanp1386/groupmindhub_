# GroupMindHub — Prototype-Driven MVP (Now)

This bundle is built from your **latest canvas prototype** and includes agent-facing docs.

## Contents
- `prototype.html` — single-file UI prototype (latest from canvas).
- `AI_CONTEXT.md` — compact context for Copilot/agents.
- `SPEC.md` — MVP specification v0.3 (concise).
- `types.d.ts` — data model interfaces.
- `COPILOT_PROMPT.md` — ready-to-paste prompts.
- `.gitignore` — basic defaults.

## Run locally
Open `prototype.html` directly in a browser, or serve it:

```bash
# Python 3
python -m http.server 8080
# or Node
npx http-server -p 8080
```

Then visit http://localhost:8080/prototype.html
